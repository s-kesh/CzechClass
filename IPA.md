## IPA for Czech Language

### Vowels

Czech has two type of vowels long and short.
Long vowels are diffentiated by acute ($/acute$) sign.

| Czech Letter | IPA Symbol | Devanagari Equivalent |
|--------------|------------|-----------------------|
| a            | /a/        | अ                     |
| á            | /aː/       | आ                     |
| e            | /ɛ/        | ए                     |
| é            | /ɛː/       | ए                     |
| i            | /i/        | इ                     |
| í            | /iː/       | ई                     |
| o            | /o/        | ओ                     |
| ó            | /oː/       | औ                     |
| u            | /u/        | उ                     |
| ú            | /uː/       | ऊ                     |
| y            | /i/        | इ                     |
| ý            | /iː/       | ई                     |
